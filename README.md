# OSS SDK for Java Developers

## Requirements

- Java 6+

## Build

```shell
git clone ...
cd aliyun-oss-java-sdk
mvn clean packge -DskipTests
```

- to run function tests, you will have to config user account in src/test/java/com/aliyun/oss/integrationtests/TestConfig.java,
